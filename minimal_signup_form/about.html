Minimal sign up form

Form and error checking done with pure HTML5 elements. The inputs are “halved” using pseudo-elements. Because we can’t add pseudo-elements for <input> we have to use an extra <span> for each input.

Tested on Mac (Chrome 30, Firefox, Safari), Chrome, Firefox, IE 8-9-10, iOS 7, and WP 8 (IE 10). Support for older browsers was added using JavaScript.

For the password strength indicator, I’m using JS to add classes that specify the colour and animations using CSS3.

No form data is stored, or passed over to the NSA. Inspired by iCloud’s login form.
